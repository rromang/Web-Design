Weather Visualizations: Latitude vs X

Summary

This page is to showcase the analysis performed to show the weather changes for different cities as you go closer to the Equator. Using the OpenWeatherMapAPI, weather information for several cities (~500) was obtained and dataframes containing information pertaining to maximum temperature, cloudiness, wind speed and humidity.

Using Seaborn to plot the data, the information collected was visualized to identify patterns among the data compared.
